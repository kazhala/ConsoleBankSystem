All files are stored in bin/debug/netcoreapp2.1
All the account number are stored in a file named "accDB.txt"
All the 0 stored in "accDB.txt" is placeholders for the deleted account, please don't alter them

The program comes with a few account number pre-created
If you would like to test program with empty accounts, you could clear the accDB.txt
Deleting all <account number>.txt is not necessary, it would be override when new accounts is created

The email usually takes a few seconds to send, please wait patiently

To run the program, just press run button
